Who I Am

It's nice to meet you, I'm Joseph Chretien-Golden. I develop thoughtful digital experiences from the front to back end.

I’m a passionate software engineer with 1-2 years of industry experience in full stack development, front-end development and user experience in Agile team environments.

In my off time, I am a developer, minimal-design lover, book worm, piano player, girl dad, film-maker, video game fan and learning enthusiast.

Discover more about my skills in detail below.

Full-Stack Development

Developed web apps using React, Node.js, Express, GraphQL, Vue.js, HTML5 and CSS3

Modernized and built front end reusable components for Zuul.

Used Asynchronous Reactive programming with Vue.js using observables to handle data streams from the server to the client.

Created Zuul's transaction reporting infrastructure and module using API’s (REST), Vue.js and Bootstrap which allows a user to view and download relevant transaction reports within brand/restaurant network. Also created Zuul’s CMS which allows a user to update menus, menu items, and more.

Frontend Development

Developed websites using React, Vue, HTML5, JavaScript, and CSS3.

Collaborated with The Room Place's UX team to redesign, create templates and set company wide standards for software usability and user experience.

Integrated git and AWS devops pipeline for web application.

UI/UX Design

Utilize Figma design system to create low and hi-fidelity, wireframes, user flows, competitive analysis, and conduct both quantitative and qualitative research to increase impact for stakeholders.
